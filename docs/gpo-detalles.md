# GPO aplicadas a Alumnos

### Bloqueo de Símbolo del sistema (CMD)
Se configuró una política para deshabilitar el acceso al símbolo del sistema,
evitando la ejecución de comandos no autorizados por parte de los alumnos.

### Bloqueo de almacenamiento extraíble
Se aplicó una política que restringe el uso de dispositivos USB,
reduciendo el riesgo de malware y fuga de información.

Ambas políticas fueron vinculadas a la OU Alumnos.
