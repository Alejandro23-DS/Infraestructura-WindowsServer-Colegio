## Arquitectura del sistema

La infraestructura está compuesta por un servidor Windows Server 2022
con rol de controlador de dominio, y un equipo cliente unido al dominio.

#### Servidor:
- IP: 192.168.10.10
- Roles: AD DS, DNS, DHCP

#### Cliente:
- Sistema Windows
- Unido al dominio para pruebas de políticas

La arquitectura general se muestra en el diagrama ubicado en /diagrams.
